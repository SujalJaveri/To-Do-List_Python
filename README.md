# To-Do List Application (Python CLI)

A simple command-line based To-Do List application built using Python.  
This project demonstrates how to manage tasks with persistent storage using file handling.

---

## Features
- View all tasks
- Add new tasks
- Delete existing tasks
- Menu-driven command-line interface
- Tasks are saved permanently using a text file
- Handles invalid inputs gracefully

---

## Concepts Used
- Python Lists
- File Handling (`read`, `write`)
- Functions & Modular Programming
- Loops (`while`)
- Conditional Statements
- Exception Handling (`try-except`)
- String Formatting (f-strings)

---

## Project Structure

- `todo_app.py` → Main application file  
- `tasks.txt` → Stores tasks persistently  
- `README.md` → Project documentation  

---

## How to Run
Make sure Python is installed on your system.

```bash
python todo_app.py
```
